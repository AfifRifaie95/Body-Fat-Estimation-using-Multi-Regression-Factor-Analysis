# Body Fat Estimation using Multi Regression & Factor Analysis in SPSS  

![ezgif-6-a6c584d0d8](https://github.com/AfifRifaie95/SPSS-Multivariate-Methods-for-Data-Analysis/assets/159521904/cfe221ec-39ca-42ea-bf7d-8eace4887cbe)


The purpose of this project is to create a prediction model for body fat association and quantification by applying multi regression analysis with a stepwise method. It is based on a set of metric independent variables such as biometrics, anthropometrics, and physiological measurements.
